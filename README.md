# TheSynLab

Vite + React + TypeScript app with shadcn/ui and **Convex** (backend). Supabase has been removed.

## Run locally

```bash
npm install
cp .env.example .env   # then set CONVEX_DEPLOY_KEY if using npx convex dev
npm run dev
```

App runs at **http://localhost:8080/**.

## Convex

Backend is Convex (deployment `kindhearted-lark-661`). See [CONVEX_DEPLOY.md](./CONVEX_DEPLOY.md) for credentials, env vars, and Coolify deployment.
